## Big Oh Notation
O : same or faster<br>
o : faster but not the same

θ : same rate

Ω : same or slower<br>
ω : slower but not the same


## Example
n^(4/3) = O(nlogn) --> False <br>

3n^3 + 4n^2 + 5n + 6 = θ(n^3) --> True <br>

n(n-1)/2 = O(n^2) --> True  <br>

2^n = ω(n) --> True <br>

n^3 = O(n^2) --> False <br>

n^2 = O(n^3) -- > True <br>
